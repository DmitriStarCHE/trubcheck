# PIPE TRACKER PRO — Проект учёта труб

## Обзор директории

Это **директория планирования и референсов** для мобильного PWA-приложения **Pipe Tracker Pro** — приложения для учёта отгрузки/приёма стальных труб с калькулятором тоннажа. Проект ещё не начат — код отсутствует.

## Содержание плана разработки

### Технический стек
- **Платформа**: PWA на React 18, Vite, offline-режим
- **Хранение**: IndexedDB, все данные локально, без сервера
- **Библиотеки**: Framer Motion, jsPDF (печать), XLSX (экспорт)

### Дизайн-система
- Цвета: `#2C3531` (фон), `#116466` (акцент teal), `#D9B08C` (золото), `#FFCB9A` (хайлайт), `#D1E8E2` (текст)
- Стиль: gradient-кнопки, inset-shadow поля, backdrop-blur тултипы, spring-анимации

### Страницы приложения
1. **Калькулятор труб** — выбор ГОСТа, диаметра, толщины; расчёт веса по формуле `W = 0.02466 × S × (D − S)`; поддержка ГОСТ 8732, 8734, 10704/10705, 20295, 3262, 633 и опционально API 5L
2. **Учёт отгрузки/приёма** — шапка документа (тип, дата, контрагент, авто), таблица длин труб, автоподсчёт количества, общей длины и тоннажа
3. **История** — список карточек, фильтры, поиск, экспорт в PDF/Excel

### Этапы (5 сессий)
1. Базовая структура, роутинг, дизайн-система
2. Калькулятор (ГОСТ-данные, формулы, селекторы)
3. Учёт (форма, таблица длин, автоподсчёт)
4. История (список, фильтры, просмотр)
5. Экспорт PDF/Excel, PWA manifest, offline-режим

## Использование

Данная директория служит отправной точкой для реализации приложения. Следующий шаг — создание React-проекта (Vite) и реализация по плану из `pipe_app_dev_plan.html`.
